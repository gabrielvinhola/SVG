Gradiente

O elemento <linearGradient> faz um gradiente que muda as cores ao longo de uma linha. Podemos definir um gradiente linear como horizontal, vertical ou angular.
O gradiente é usado sempre dentro do elemento <defs>, isso para que você possa referenciar no seu SVG.
O elemento <radialGradient> faz um gradiente que muda as cores ao longo de raio.


Atributos

y1 e y2 devem ser igual e x1 e x2 devem ser diferentes para que seja horizontal o gradiente linear.
x1 e x2 são iguais e y1 e y2 são diferentes para que seja vertical o gradiente linear.
x1, x2, y1 e y2 são diferentes para que seja angular o gradiente linar.
id: você usa ID para referenciar o seu gradiente.
cx e cy: são usados para centralizar o gradiente radial, especificando as porcentagens que você quer.
fx e fy: são usados como pontos focais para o gradiente radial, especificando as porcentagens que você quer.
r: raio do gradiente.

<stop> é onde você define o range de cores do gradiente, sendo ele feito por duas ou mais cores.
O atributo offset define quando a cor do gradiente irá começar e quando terminar;
O atributo stop-color indica qual cor usar na parada do gradiente;
O atributo stop-opacity define a opacidade da cor dada na parada do gradiente (0 ou 1);
O atributo fill faz a ligação com o que foi definido na tag <defs>, ou seja, nossa cor do gradiente para o elemento que foi criado e nele será aplicado o gradiente da cor desejada.
