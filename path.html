Path

O elemento <path> cria uma forma compostas por linhas não necessariamente conectadas como polyline, mas pode ser composto tanto de retas quanto de curvas, pode ser uma forma aberta ou fechada.


Atributos

Os seguintes comandos estão disponíveis para uso na tag path:

M = moveto: move o cursor para a posição especificada, sem desenhar;
L = lineto: cria uma linha reta do ponto imediatamente anterior até o ponto especificado;
H = horizontal lineto: cria uma linha horizontal do ponto imediatamente anterior até o ponto especificado;
V = vertical lineto: cria uma linha vertical até o ponto especificado;
C = curveto: cria uma curva até o ponto especificado;
S = smooth curveto: cria uma curva suave até o ponto especificado;
Q = quadratic Bézier curve: cria uma curva quadrática até o ponto especificado;
T = smooth quadratic Bézier curveto: cria uma curva quadrática suave até o ponto especificado;
A = elliptical Arc: cria um arco elíptico até o ponto especificado.
Z = closepath


Nota: Todos os comandos acima também podem ser expressados com letras minúsculas. Letras maiúsculas indicam posição absoluta; letras minúsculas indicam posição relativa.
